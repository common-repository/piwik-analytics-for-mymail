== Description ==

> This Plugin requires [MyMail Newsletter Plugin for WordPress](https://evp.to/mymail?utm_campaign=wporg&utm_source=Piwik+Analytics+for+MyMail)

Integrates Piwik Analytics with MyMail Newsletter Plugin to track your clicks
Requires the MyMail Newsletter plugin >= 2.0 and the Piwik Analytics tracking code on the website

== Screenshots ==


== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
2. Activate the plugin
